# Wireframr-TR

Updated the original plugin with Comic Neue font.

---
The Comic Neue font must be installed in your Mac's FontBook app for this plugin to work properly. 

First, download the [Comic Neue font](Comic_Neue.zip) and install it into FontBook.<br>
Next, download the [Wireframr-TR Sketch plugin](wireframr.sketchplugin.zip) and install in Sketch.
